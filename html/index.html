Три честные вещи о текущем состоянии

Никаких маркетинговых формулировок — только то, что есть на самом деле.

Инструкций пока нет

Кодовая база и структура проекта меняются каждый день, поэтому подробное руководство по установке появится только когда архитектура устоится.

Без гарантий стабильности

Проект не готов для конечного пользователя. Что-то может падать, ломаться или быть недописанным — это нормальное состояние на этом этапе.

Для любопытных

Исходный код открыт для изучения. Но «из коробки» пока ничего стабильно не запускается — и мы предупреждаем об этом заранее.
